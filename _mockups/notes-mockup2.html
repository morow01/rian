How it works — summary
List view: Compact rows — coloured dot + title + location + date + priority badge. Nothing else until you need it.
Expand: Tap any row — arrow rotates, detail area slides open. Description is editable inline. Changes auto-save (same as timesheet).
Filter chips: Tap High / Medium / Low / Done to filter the list instantly.
Add: "+ Add" button creates a new blank note at the top, auto-expanded and ready to type.
Mark done: Button inside the expanded detail. Moves note to Done section, strikes through.
Delete: Button inside expanded detail (with confirmation).
Sync: Same field-level merge as the timesheet. Works across all devices.
