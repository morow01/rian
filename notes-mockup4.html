Theme note
The mockup is shown in the default light theme. In the real build every colour will use CSS variables (var(--bg-card), var(--text-primary), etc.) exactly as the rest of the app does — so switching to dark, slate, retro or any other theme will automatically apply to the Notes tab too, with no extra work.

🔔 Remind button — visible in screen ②, greyed out. Will show a toast "Reminders coming soon" when tapped until the feature is built in the next stage.
