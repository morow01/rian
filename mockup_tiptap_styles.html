DC Power Load Test

Procedure Notes

  1. Isolate battery bank from rectifier bus
  2. Apply constant current drain at 0.1C for 10 minutes
  3. Record terminal voltage every 60 seconds
  4. Reconnect and verify float voltage recovery within 15 min
// Load test measurement log
const readings = [
  { t: 0,  v: 2.21, a: 4.8 },
  { t: 60, v: 2.19, a: 4.7 },
  { t: 120,v: 2.18, a: 4.8 },
];
console.log('Test complete');

Task List

  • Schedule return visit after delivery

Additional Notes

  • Fire extinguisher tag checked — expires Nov 2026
  • AC supply healthy, no UPS bypass alarms

Other style improvements I'd suggest
New
Status colours in table cells

Green/amber/red tint for OK/Warn/Bad cells. Makes scanning voltage tables instant — shown above in the mockup.

Tweak
Blockquote background

Add a light blue-gray tint (#F8FAFC) so alarm/callout notes stand out from body text. Currently plain white.

Tweak
Code block dark theme

Darker code blocks (#1E293B) with syntax colour hints. Current code blocks are light gray — harder to distinguish from body.

Tweak
Inline code border

Add a subtle 1px border (#E2E8F0) around inline code. Stops code from blending into body text.

New
Horizontal rule style

A clean 1px line (#E8E6E0) to visually separate note sections. Useful for long site visit notes.

Tweak
Ordered list markers

Muted gray (#94A3B8) markers with slightly smaller weight. Keeps numbered procedures readable without being noisy.
