Journal UX — Add Content Redesign

4 options for improving "Add Notebook / Add Section / Add Page" UX

Option A — Header Buttons
Each column header has a small + button on the right. Add Notebook lives in the notebooks header. Add Section moves to the notebooks header too (adds a section to the selected notebook). Add Page lives in the pages header. No buttons at the bottom — the footer row is gone completely.

Option B — Hover Context

Option C — Notion Style (New at top)
