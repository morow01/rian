Journal Transitions
Compare the three approaches
View Transitions API — Browser takes a snapshot of the old screen, renders the new one, then crossfade-slides between them. 5 lines of JS. Chrome 111+ / Samsung Browser.
Two-panel slide — Both screens live side-by-side in an oversized container. Navigation translates the container left/right. Classic iOS-style. Works everywhere.
Clone + overlay — Before render, the old screen is cloned and overlaid. New content renders behind it. Clone slides out while new content slides in. Works everywhere.
