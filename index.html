01 — Timesheet · Summary

Your week at a glance

The weekly summary rolls up every task, code, and hour into a clean overview. See ordinary vs overtime totals per day and for the full week — then email the whole thing in one tap.

  • Day-by-day breakdown Ordinary and overtime hours listed for each day of the week, with running totals.
  • Overtime highlighted OT hours stand out in amber so you can spot them immediately.
  • One-tap email Send a formatted summary straight from the app — no copy-paste, no spreadsheet needed.
  • Code totals Hours grouped by work code so supervisors can see exactly where time was spent.

02 — Notes

Field notes that actually keep up

Capture anything that needs following up — safety issues, reminders, punch list items. Prioritise, filter, and mark done when resolved.

  • Priority levels Tag notes as Urgent, High, or Normal. Red left border makes critical items impossible to miss.
  • Filter & focus Switch between All, Open, and Done with one tap. No clutter.
  • Bulk actions Select multiple notes and mark them done or delete in one go.
  • Rich content Each note supports a full description body, not just a title.
